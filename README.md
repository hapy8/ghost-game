# Ghost Run - Next Gen

A spooky endless runner game built with Python and Pygame!

## Features
- **Next-Gen Graphics**: Particle effects, smooth animations, and dynamic backgrounds.
- **Difficulty Scaling**: The game gets faster as you play.
- **High Scores**: Compete against yourself.

## How to Play
1. Run `python ghost_run_game.py`
2. Press **SPACE** to Jump.
3. Avoid Obstacles and collect Orbs!
